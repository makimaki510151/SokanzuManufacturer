相関図エディタ

soukanzu と互換の JSON 形式。SVG ではアイコンにマウスオーバーでプロフィールを表示できます。

データ出力

ヒント

  • 盤面上でマウスホイールを回すと拡大・縮小できます（表示リセットで全体表示に戻ります）。
  • 選択ツールのとき、要素のない余白を左ドラッグすると盤面をパン（表示位置を移動）できます。
  • テキストはダブルクリックまたは右クリックで内容・色・サイズを変更できます。
  • 画像は右クリックでプロフィールを編集できます。リンクURL（または詳細に URL のみ）を入れると、SVG 書き出しで画像クリックでそのページを開けます。
  • グループ枠は選択ツールでドラッグして移動。青い角・辺のつかみで拡大縮小できます（離すと範囲内の要素で members を更新）。グループは右クリック/ダブルクリックで装飾を編集できます。
  • 接続ツールで始点・終点の順にクリックし、ダイアログで「沿線の文字」を設定できます。文字は線に沿って表示され、上下が逆にならない向きに自動調整されます。既存の線は右クリックまたはダブルクリックで編集できます（クリック判定は少し広めです）。
  • JSON の画像は data: URL または相対パスに対応します。
  • SVG はブラウザで開くだけで、画像上にマウスを乗せるとプロフィールが表示されます。
  • PNG は SVG と同じ範囲（図の内容＋周囲の余白）のビットマップとして書き出します（透過なし・ホバーなし）。
